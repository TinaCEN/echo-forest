## 🚀 使用建议

### 第一次使用
```bash
cd echo_garden
python3 echo_garden_realtime.py
```

### 启用完整音频功能
```bash
# macOS用户
bash install_audio.sh

# 然后重新运行
python3 echo_garden_realtime.py
```

### 开始创作
1. 点击"🎤 开始实时"启用麦克风
2. 观察左侧波形显示
3. 发出不同声音（说话、唱歌、乐器等）
4. 点击画布种植音频响应的树木
5. 尝试不同主题和保存作品
